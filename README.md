# Legal Information Center

A comprehensive legal documentation website for our mobile applications distributed on Google Play Store, with full compliance to GDPR, CCPA, and Google Play policies.

**Last Updated:** January 31, 2026

## Overview

This project provides a centralized hub for all legal documentation related to mobile applications. It ensures transparency and compliance with international data protection regulations and Google Play Store requirements.

## Project Structure

```
legal/
├── index.html          # Main landing page with links to all legal documents
├── privacy.html        # Privacy Policy - Data collection and user rights
├── terms.html          # Terms of Use - Application usage terms
├── support.html        # Support Center - Contact and assistance information
├── css/
│   └── style.css       # Styling for all HTML pages
└── README.md           # This file
```

## Compliance Standards

This documentation ensures compliance with:

| Standard | Coverage |
|----------|----------|
| **GDPR** | General Data Protection Regulation for EU users |
| **CCPA** | California Consumer Privacy Act for US users |
| **Google Play Policies** | Developer Program Policies and Data Safety |
| **Data Protection** | User rights, data retention, and security measures |
